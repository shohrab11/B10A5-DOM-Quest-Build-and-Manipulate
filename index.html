What is document object model (DOM) ?

Document object model is a process of manipulate HTML. Thorough DOM we can create HTML TAG.Add class on the HTML Tag Remove class on the HTML.

How do you select an element from the DOM ?

JavaScript provides several methods to select DOM elements. Here are some of them

  1. Using getElementById
  2. Using getElementsByClassName
  3. Using Document.querySelector
  4. Using Document.querySelectorAll
What is the event delegation in the context of DOM, and why is it useful ?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows us to handle events at a higher level in the DOM tree other than the level where the event was first received

How do you manipulate an element's attributes and style using DOM ?

The classList object only provide methods to change the class of an element. If we want to change other attributes like id, href, or src, you can use the setAttribute() method. The setAttribute() method accepts two arguments:The value of the attribute to set
